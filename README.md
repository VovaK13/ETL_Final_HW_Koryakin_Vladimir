# ETL_Final_HW_Koryakin_Vladimir

Финальное ДЗ по курсу ETL.

В проекте реализованы 4 аналитические витрины, которые построены на основе данных, реплицированных из MongoDB в PostgreSQL. Необходимые для проверки скрины приложены в репозиторий, описание витрин ниже.


Витрина 1: Активность пользователей (mart.user_activity)
Витрина предназначена для поведенческого анализа пользователей - позволяет понять, как пользователи взаимодействуют с сервисом, какие страницы посещают, какие действия совершают и сколько времени проводят на сайте.

date	DATE	Дата активности (партиция)

user_id	VARCHAR(50)	Идентификатор пользователя

total_sessions	INT	Количество сессий пользователя за день

avg_session_duration_minutes	NUMERIC(10,2)	Средняя продолжительность сессии в минутах

total_pages_visited	INT	Общее количество посещенных страниц (с учетом повторов)

unique_pages_visited	INT	Количество уникальных страниц

total_actions	INT	Общее количество действий пользователя

most_frequent_action	VARCHAR(50)	Самое частое действие пользователя

devices_used	TEXT[]	Массив устройств, которые использовал пользователь

last_activity	TIMESTAMP	Время последней активности

updated_at	TIMESTAMP	Время последнего обновления витрины


Витрина 2: Эффективность работы поддержки (mart.support_efficiency)
Витрина отслеживает ключевые метрики работы службы поддержки, позволяет анализировать нагрузку и качество обслуживания клиентов.

date	DATE	Дата создания тикетов

issue_type	VARCHAR(50)	Тип проблемы (payment, delivery, technical и т.д.)

total_tickets	INT	Общее количество созданных тикетов

open_tickets	INT	Количество открытых тикетов

closed_tickets	INT	Количество закрытых тикетов

avg_resolution_time_hours	NUMERIC(10,2)	Среднее время решения проблемы в часах

max_resolution_time_hours	NUMERIC(10,2)	Максимальное время решения

min_resolution_time_hours	NUMERIC(10,2)	Минимальное время решения

total_messages	INT	Общее количество сообщений в тикетах

unique_users	INT	Количество уникальных пользователей, обратившихся в поддержку

updated_at	TIMESTAMP	Время последнего обновления витрины


Витрина 3: Статистика модерации (mart.moderation_stats)
Витрина анализирует процесс модерации отзывов пользователей, помогает отслеживать качество контента и эффективность модераторов.

date	DATE	Дата подачи отзыва

moderation_status	VARCHAR(20)	Статус модерации (pending, approved, rejected)

total_reviews	INT	Общее количество отзывов

avg_rating	NUMERIC(3,2)	Средняя оценка в отзывах

total_flags	INT	Общее количество флагов (отметок о проблемах)

unique_products	INT	Количество уникальных товаров, на которые оставили отзывы

unique_users	INT	Количество уникальных пользователей, оставивших отзывы

updated_at	TIMESTAMP	Время последнего обновления витрины


Витрина 4: Популярность товаров (mart.product_popularity)
Витрина собирает статистику по товарам: количество просмотров, отзывов, рекомендаций. Помогает понять, какие товары наиболее востребованы.

date	DATE	Дата

product_id	VARCHAR(50)	Идентификатор товара

total_views	INT	Количество просмотров страницы товара

total_reviews	INT	Количество отзывов на товар

avg_rating	NUMERIC(3,2)	Средняя оценка товара

times_recommended	INT	Сколько раз товар был рекомендован пользователям

updated_at	TIMESTAMP	Время последнего обновления витрины
